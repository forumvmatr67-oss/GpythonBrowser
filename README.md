# 🐍 GPython — умный поиск без слежки

**GPython** — это веб-поисковик с аккаунтами, тёмной темой, историей поиска и реальными результатами через DuckDuckGo API.  
Создан как альтернатива Google/Яндекс с фокусом на приватность и стиль.

## ✨ Возможности

- 📝 Регистрация / Логин (безопасное хеширование паролей)
- 🔎 Поиск через DuckDuckGo (честные результаты)
- 📜 Личная история поиска
- 🎨 Переключение тёмной/светлой темы
- ⚡ Живые подсказки при вводе
- 🗑️ Удаление аккаунта вместе со всей историей
- 📱 Адаптивный дизайн
